WordPress/WooCommerce plugin to provide pretty filenames for downloads and improve the API for our needs.

== Description ==
Names the download file (in Content-Disposition) like the download name.
Add API hooks to include full variations in products list.

== Installation ==
Standard installation.

Will only work with WooCommerce.
